What it does

Everything a castable mold needs - generated in one click.

No more hand-modelling parting surfaces, sprues and registration keys. Pick your mesh, dial the options, and get clean, watertight, printable parts.

Pour box or direct mold

Print a thin-walled jacket and pour liquid silicone into the gap, or print the mold and cast resin / wax / plaster straight in. Optional glove-mold skin keys for a thin flexible skin.

Tray molds for flat objects

A one-part open pan for text, logos, coins and relief tiles: embed the object and pour silicone for a stamp, or frame a real object. Rectangular or material-saving hugging outline, auto-laid flat.

Splits that self-register

Auto-oriented contoured parting that follows the model, cone keys or interlocking teeth, and full-height clamp wings with bolt holes that hug the body and run up the funnel.

Real pour funnels

A proper raised spout - adjustable throat radius, height and mouth flare - bored clean through into the cavity, with a circular opening and silicone clearance over the model.

Multi-part for undercuts

Shapes that undercut on every side? Split into 3-4 radial wedges that each pull straight out, with seam pins or per-seam bolted flanges. Auto-orientation picks the cleanest release.

Detachable keyed plate

A separate base plate that registers the model in a pocket and the shell in a tongue-and-groove - with a dial-in fit clearance so the printed parts actually click together.

Know your pour

Live silicone, cast and printed-plastic volume and weight estimates - in ml and grams, with material density presets - so you mix the right amount before you commit.

How it works

Four steps from mesh to mould.

Select your mesh

Any closed-ish model - sculpt, scan or download. MoldForge heals messy and heavy meshes for you.

Dial it in

Mold type, split axis, funnel, base plate, clearances - or just keep the smart defaults.

Generate

One click produces clean, watertight, registered mold parts on top of your model.

Print & pour

Export STL, print, then pour silicone or cast directly. The estimates told you how much to mix.

Download for Blender
Blender 5.1+ · Windows · macOS · Linux
Beta software. v0.23.2 is an early release, so many bugs are still possible. Save a backup of your .blend before generating, and please report what breaks.
  1. In Blender: Edit ▸ Preferences ▸ Get Extensions ▸ ⌄ ▸ Install from Disk… and pick the zip.
  2. Make sure MoldForge is enabled.
  3. Open the 3D viewport sidebar (N) → the MoldForge tab.
  4. Select your model, set options, hit Generate Mold.

Get auto-updates (optional)

Add MoldForge as a Blender extension repository once, and Blender flags new versions with a one-click Update — no re-downloading.

  1. Edit ▸ Preferences ▸ Get Extensions ▸ Repositories ▸ ＋ ▸ Add Remote Repository
  2. Paste this URL, tick Enabled, and confirm:
https://moldforge.plesuro.eu/downloads/index.json

Then hit Check for Updates (the ⟳ in that menu). New releases show an Update button from then on.
